# Plan de Pruebas

## Pruebas funcionales

- Login por rol.
- Registro de producto por vendedor.
- Aprobacion de producto por admin.
- Compra con stock suficiente.
- Bloqueo de compra si supera stock.
- Cupon aplicado al checkout.
- Reseña posterior a compra.
- Penalizacion y veto de vendedor.
- Registro, busqueda, actualizacion y eliminacion de articulos.
- Registro de proveedor y orden de compra.

## Pruebas de validacion

- Formularios requeridos.
- Metodo de pago Yape/Plin con operacion.
- Metodo tarjeta con datos obligatorios.
- Metodo mixto con suma equivalente al total.

## Mejoras futuras

- Pruebas automatizadas E2E.
- Conexion real a Cloud SQL.
- Autenticacion segura con contraseñas cifradas.
