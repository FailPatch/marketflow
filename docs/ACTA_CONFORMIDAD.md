- FAQ/consultas frecuentes.
- Manual de usuario.
- Manual tecnico.

- Evidencia de codificacion por modulos.

Observacion: la version actual usa datos en memoria para facilitar la presentacion. Para produccion se puede conectar a Cloud SQL MySQL usando las variables documentadas en el README.
