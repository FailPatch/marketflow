# Acta de inicio

Proyecto: NEXORA - Marketplace para Desarrollo Web Integrado.

Objetivo: implementar una aplicacion web para compra y venta de productos con roles, catalogo, carrito, ordenes de compra, administracion, reseñas, penalizaciones, dashboard y despliegue cloud.

Alcance:

- Registro e inicio de sesion por rol.
- Gestion de empleados, proveedores, articulos y ordenes de compra.
- Publicacion de productos por vendedores.
- Aprobacion de publicaciones por administrador.
- Compra de productos con control de stock, cupones, comprobantes y metodos de pago.
- Reseñas de compradores para apoyar la supervision del vendedor.
- Dashboard, auditoria y documentacion tecnica.

Fecha: 07 de julio de 2026.
