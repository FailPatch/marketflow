# Manual de Usuario

## Comprador

1. Inicia sesion como comprador.
2. Revisa el catalogo.
3. Agrega productos al carrito.
4. Aplica cupones si tiene disponibles.
5. Elige comprobante y metodo de pago.
6. Confirma la compra.
7. Registra una reseña del producto recibido.

## Vendedor

1. Inicia sesion como vendedor.
2. Publica productos con nombre, categoria, precio, stock e imagen.
3. Espera aprobacion del administrador.
4. Revisa penalizaciones si existieran.

## Administrador

1. Inicia sesion como administrador.
2. Revisa dashboard.
3. Aprueba o pausa productos.
4. Revisa reseñas.
5. Penaliza o veta vendedores segun incidencias.

## Empleado

1. Accede al modulo de gestion.
2. Registra proveedores.
3. Registra articulos.
4. Genera ordenes de compra.
