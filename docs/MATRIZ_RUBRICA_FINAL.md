# Matriz de cobertura de rubrica final

| Criterio | Donde se evidencia |
| --- | --- |
| Presentacion en distintos dispositivos | Vistas HTML5 responsivas y CSS en `static/css/styles.css` |
| Consultas del cliente | Ruta `/faq` con preguntas frecuentes |
| Manuales | `docs/MANUAL_USUARIO.md` y `docs/MANUAL_TECNICO.md` |
| Dashboard de gestion | Ruta `/dashboard` y panel `/admin` |
| Acceso por tipos de usuario | Login con comprador, vendedor, administrador y empleado |
| Gestion de compra/venta | Catalogo, carrito, checkout, panel vendedor y ordenes de compra |
| Grabar | Formularios de empleados, proveedores, articulos y ordenes |
| Buscar | Busqueda de articulos por nombre o ID |
| Actualizar | Actualizacion de articulos |
| Eliminar | Eliminacion de articulos |
| Rendimiento y pruebas | `docs/PLAN_PRUEBAS.md` y prueba `MarketplaceRulesTest` |
| Actas | `docs/ACTA_INICIO.md` y `docs/ACTA_CONFORMIDAD.md` |
| Aplicaciones/patrones | `docs/EXPLICACION_MVC_DAO.md` |
| Codificacion por modulos | `docs/EXPLICACION_CODIFICACION.md` |
| Cloud | `Dockerfile`, `cloudbuild.yaml`, `.gcloudignore` y README |
